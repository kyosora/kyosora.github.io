常用網路通訊埠

發表於 2024-01-26 更新於 2024-11-27 分類於 網路

你不可不知的網路通訊埠大全

在現代網路世界中，通訊埠（Port）就像是建築物的門牌號碼，它們引導著網路封包準確地到達目的地。無論你是網路工程師、系統管理員，還是對網路技術感興趣的使用者，了解常用的通訊埠都是必備的基礎知識。

什麼是通訊埠？

通訊埠是網路通訊中的虛擬概念，用於區分電腦中執行的不同網路服務。每個通訊埠都有一個 0-65535 之間的數字，而最常用的服務通常會使用固定的通訊埠號碼。

常用通訊埠分類指南

檔案傳輸相關通訊埠

  • FTP（檔案傳輸協定）
    • 20：FTP 資料傳輸埠
    • 21：FTP 控制埠
    • 989：FTPS 資料傳輸埠（加密版本）
    • 990：FTPS 控制埠（加密版本）
    • 115：SFTP（SSH 檔案傳輸協定）

遠端連線通訊埠

  • 22：SSH/SCP

    • 安全的遠端連線和檔案傳輸
    • 現代系統的標準遠端管理工具

  • 23：Telnet

    • 傳統的遠端連線協定
    • 注意：不加密，建議改用 SSH

  • 3389：Windows Remote Desktop

    • Windows 遠端桌面連線
    • 提供圖形化的遠端操作介面

網路基礎服務通訊埠

  • 53：DNS

    • 網域名稱解析服務
    • 將網域名稱轉換為 IP 位址

  • 67/68：DHCP

    • 67：DHCP 伺服器
    • 68：DHCP 客戶端
    • 自動分配 IP 位址的服務

  • 80：HTTP

    • 網頁瀏覽的基本協定
    • 未加密的網頁傳輸

  • 443：HTTPS

    • 加密的網頁傳輸
    • 安全性較高，現代網站標配

郵件相關通訊埠

  • 25：SMTP

    • 郵件寄送協定
    • 用於發送電子郵件

  • 110：POP3

    • 郵件接收協定
    • 用於下載電子郵件

  • 143：IMAP

    • 進階郵件協定
    • 支援郵件同步和管理

資料庫服務通訊埠

  • 1433：SQL Server

    • Microsoft SQL Server 的預設通訊埠
    • 企業級資料庫服務

  • 1434：Azure DMS

    • Azure 資料庫遷移服務
    • 用於雲端資料庫遷移

  • 3306：MySQL

    • MySQL 資料庫的預設通訊埠
    • 最流行的開源資料庫系統

  • 5022：SQL Server 可用性群組

    • SQL Server 叢集服務
    • 用於高可用性設定

其他重要服務通訊埠

  • 119：NNTP

    • 網路新聞傳輸協定
    • 用於新聞組服務

  • 123：NTP

    • 網路時間協定
    • 用於時間同步

  • 137/138/139：NetBIOS

    • 137：名稱服務
    • 138：資料廣播
    • 139：連線服務
    • Windows 網路共享必備

  • 389：LDAP

    • 輕量級目錄存取協定
    • 用於集中式身分驗證

安全性建議

  1. 定期檢查開放的通訊埠
  2. 關閉不必要的服務和通訊埠
  3. 使用防火牆限制通訊埠存取
  4. 優先選用加密版本的協定（如 HTTPS 替代 HTTP）
  5. 定期更新系統和服務以修補安全漏洞

總結

了解這些常用通訊埠不僅有助於網路疑難排解，也是資安防護的重要基礎。建議將此清單作為參考指南，在日常工作中靈活運用。記住，安全永遠是首要考慮，確保只開放必要的通訊埠，並時刻關注最新的安全建議。
