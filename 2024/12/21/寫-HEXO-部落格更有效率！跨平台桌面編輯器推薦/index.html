寫 HEXO 部落格更有效率！跨平台桌面編輯器推薦

2024-12-21T10:30:00.000Z2025-01-02T05:55:55.963Z台中

你有遇過這些寫 HEXO 部落格的小困擾嗎？

  • 一邊寫文章一邊要切換視窗預覽效果
  • 想要快速修改文章的標題、分類和標籤要手動改 YAML
  • 分類和標籤越來越多，不知道之前用過哪些
  • 想要備份所有文章，還要自己一個一個找檔案

如果有類似困擾的話，不妨試試看這個專為 HEXO 部落客設計的桌面編輯器 - HEXO Editor Desktop！

🌟 HEXO Editor 有什麼特別的？

這是一款開源的跨平台編輯器，專注於改善 HEXO 部落客的寫作體驗。透過圖形化介面，讓你可以更專注在內容創作上。

主要功能

  1. 一站式編輯環境

    • Markdown 編輯器配合即時預覽功能
    • 圖形化設定 Front Matter
    • 分割畫面設計，編輯和預覽一次搞定

  2. 分類標籤管理

    • 支援多層級分類
    • 顯示標籤使用統計
    • 自動完成建議功能

  3. 實用功能

    • 支援快速鍵（如 Ctrl+S 儲存）
    • 可搜尋過濾文章
    • 一鍵備份匯出

  4. 支援多種作業系統

    • Windows
    • macOS
    • Linux

💡 實際操作介紹

1. 寫作流程

開啟程式後：

  • 選擇你的 HEXO 部落格資料夾
  • 在左側列表中尋找或搜尋文章
  • 使用右側編輯區修改內容
  • 同時預覽文章呈現效果

2. Front Matter 設定

使用圖形化介面輕鬆設定：

  • 文章標題
  • 發布時間
  • 封面圖片
  • 分類和標籤
  • 其他自訂欄位

3. 分類標籤整理

完整的管理功能：

  • 檢視所有分類和標籤
  • 查看使用次數統計
  • 合併或重新命名
  • 管理階層式分類

4. 備份功能

簡單的備份流程：

  • 自動打包成 ZIP 檔
  • 包含文章和相關檔案
  • 方便備份和轉移

環境需求

  • Python 3.8 以上版本
  • PySide6
  • Git (選用)
