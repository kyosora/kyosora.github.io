еҰӮдҪ•еңЁ IIS дёҠиЁӯе®ҡ HSTS - жҸҗй«ҳз¶Із«ҷе®үе…ЁжҖ§

д»ҠеӨ©иҰҒе’ҢеӨ§е®¶еҲҶдә«дёҖеҖӢжҸҗй«ҳз¶Із«ҷе®үе…ЁжҖ§зҡ„йҮҚиҰҒжҠҖиЎ“пјҡHTTP Strict Transport SecurityпјҲHSTSпјүгҖӮйҖҷеҖӢжҠҖиЎ“еҸҜд»ҘиӘӘжҳҜз¶Із«ҷе®үе…Ёзҡ„дёҖеӨ§еҲ©еҷЁе‘ўпјҒ

жңҖиҝ‘пјҢе°ҲжЎҲејұй»һжҺғжҸҸзҡ„жҷӮеҖҷзҷјзҸҫдәҶдёҖеҖӢдҪҺйўЁйҡӘзҡ„е•ҸйЎҢгҖӮйҖҷеҖӢе•ҸйЎҢе°ұжҳҜпјҡ

HSTS жҳҜд»Җйәје‘ўпјҹ

HSTS е…ЁеҗҚжҳҜ HTTP Strict Transport SecurityпјҢжҳҜз”ұз¶Ійҡӣз¶Іи·Ҝе·ҘзЁӢд»»еӢҷзө„зҷјеёғзҡ„дёҖзЁ®з¶Іи·Ҝе®үе…Ёзӯ–з•ҘгҖӮж №ж“ҡз¶ӯеҹәзҷҫз§‘зҡ„иӘӘжҳҺпјҡ

HSTS зҡ„дҪңз”ЁжҳҜеј·еҲ¶зҖҸиҰҪеҷЁдҪҝз”Ё HTTPS иҲҮз¶Із«ҷйҖІиЎҢйҖҡдҝЎпјҢи—үжӯӨжёӣе°‘йҖЈз·ҡиў«еҠ«жҢҒзҡ„йўЁйҡӘгҖӮ

з°Ўе–®дҫҶиӘӘпјҢHSTS зҡ„дё»иҰҒеҠҹиғҪжңүпјҡ

  1. еј·еҲ¶е®ўжҲ¶з«ҜпјҲеҰӮзҖҸиҰҪеҷЁпјүдҪҝз”Ё HTTPS иҲҮдјәжңҚеҷЁе»әз«ӢйҖЈз·ҡ
  2. еҸҜд»Ҙжңүж•ҲйҳІзҰҰ SSL еүқйӣўж”»ж“ҠпјҲйҖҷжҳҜдёҖзЁ®дёӯй–“дәәж”»ж“Ҡе“Ұпјү

иҒҪиө·дҫҶеҫҲеҺІе®іеҗ§пјҹйӮЈйәјпјҢжҲ‘еҖ‘и©ІеҰӮдҪ•иЁӯе®ҡ HSTS е‘ўпјҹ

HSTS зҡ„иЁӯе®ҡж–№жі•

еҫҲеӨҡдәәеҸҜиғҪжңғжғіеҲ°дҪҝз”Ё web.config жӘ”жЎҲдҫҶиЁӯе®ҡпјҢеғҸйҖҷжЁЈпјҡ

дҪҶжҳҜе‘ўпјҢйҖҷеҖӢж–№жі•е…¶еҜҰдёҚеӨӘжӯЈзўәгҖӮеӣ зӮәйҖҷжЁЈиЁӯе®ҡжңғе°ҺиҮҙеңЁйқһ HTTPS еҚ”е®ҡдёӢд№ҹзҷјйҖҒ HSTS жЁҷй ӯпјҢйҖҷжҳҜдёҚжҮүи©Ізҡ„гҖӮ

жүҖд»Ҙе‘ўпјҢиҰҒж•ҷеӨ§е®¶дёҖеҖӢжӣҙеҘҪзҡ„ж–№жі•пјҢе°ұжҳҜдҪҝз”Ё IIS зҡ„ URL Rewrite жЁЎзө„дҫҶиЁӯе®ҡгҖӮ

жӯЈзўәзҡ„ HSTS иЁӯе®ҡжӯҘй©ҹ

жӯҘй©ҹдёҖпјҡиЁӯе®ҡ HTTPS йҮҚж–°е°Һеҗ‘

  1. йҰ–е…ҲпјҢи«Ӣжү“й–Ӣ IIS з®ЎзҗҶе“Ўд»Ӣйқў
  2. еңЁжӮЁзҡ„з¶Із«ҷдёӢпјҢжүҫеҲ°дёҰй»һйҒёгҖҢURL RewriteгҖҚжЁЎзө„
  3. жҺҘи‘—пјҢж–°еўһдёҖеҖӢијёе…ҘиҰҸеүҮ
  4. иЁӯе®ҡеҰӮдёӢпјҡ
    • еҗҚзЁұпјҡHTTP to HTTPS redirect
    • жЁЎејҸпјҡ(.*)
    • жўқд»¶пјҡ{HTTPS}пјҢйЎһеһӢйҒёгҖҢз¬ҰеҗҲжЁЎејҸгҖҚпјҢжЁЎејҸеЎ«гҖҢ^OFF$гҖҚ
    • еӢ•дҪңпјҡйЎһеһӢйҒёгҖҢйҮҚж–°е°Һеҗ‘гҖҚпјҢйҮҚж–°е°Һеҗ‘ URL еЎ«гҖҢhttps://{HTTP_HOST}/{R:1}гҖҚпјҢиЁҳеҫ—еӢҫйҒёгҖҢйҷ„еҠ жҹҘи©ўеӯ—дёІгҖҚпјҢйҮҚж–°е°Һеҗ‘йЎһеһӢйҒёгҖҢж°ёд№…пјҲ301пјүгҖҚ

жӯҘй©ҹдәҢпјҡиЁӯе®ҡ HSTS ијёеҮәиҰҸеүҮ

  1. йӮ„жҳҜеңЁгҖҢURL RewriteгҖҚжЁЎзө„дёӯпјҢйҖҷж¬Ўж–°еўһдёҖеҖӢијёеҮәиҰҸеүҮ
  2. иЁӯе®ҡеҰӮдёӢпјҡ
    • еҗҚзЁұпјҡAdd Strict-Transport-Security when HTTPS
    • и®Ҡж•ёеҗҚзЁұпјҡRESPONSE_Strict_Transport_Security
    • жЁЎејҸпјҡ.*
    • жўқд»¶пјҡ{HTTPS}пјҢйЎһеһӢйҒёгҖҢз¬ҰеҗҲжЁЎејҸгҖҚпјҢжЁЎејҸеЎ«гҖҢ^ON$гҖҚ
    • еӢ•дҪңпјҡйЎһеһӢйҒёгҖҢйҮҚеҜ«гҖҚпјҢеҖјеЎ«гҖҢmax-age=31536000гҖҚ


      иЁӯе®ҡе®ҢжҲҗеҫҢпјҢеҲҘеҝҳдәҶйҮҚж–°е•ҹеӢ•жӮЁзҡ„з¶Із«ҷпјҢи®“ж–°зҡ„иЁӯе®ҡз”ҹж•Ҳе–”пјҒ

йҖҷжЁЈдёҖдҫҶпјҢHSTS зҡ„иЁӯе®ҡе°ұе®ҢжҲҗе•ҰпјҒжҳҜдёҚжҳҜж„ҹиҰәеҫҲз°Ўе–®е‘ўпјҹ
