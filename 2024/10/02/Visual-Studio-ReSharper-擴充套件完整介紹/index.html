Visual Studio ReSharper ж“ҙе……еҘ—д»¶е®Ңж•ҙд»Ӣзҙ№

еңЁ Visual Studio зҡ„зңҫеӨҡж“ҙе……еҘ—д»¶дёӯпјҢReSharper еҸҜиӘӘжҳҜжңҖеҸ—жӯЎиҝҺзҡ„з”ҹз”ўеҠӣе·Ҙе…·д№ӢдёҖгҖӮжң¬ж–Үе°Үи©ізҙ°д»Ӣзҙ№ ReSharper зҡ„дё»иҰҒеҠҹиғҪпјҢи®“дҪ зҡ„й–Ӣзҷјж•ҲзҺҮжӣҙдёҠдёҖеұӨжЁ“пјҒ

д»ҖйәјжҳҜ ReSharperпјҹ

ReSharper жҳҜз”ұ JetBrains е…¬еҸёй–Ӣзҷјзҡ„ Visual Studio ж“ҙе……еҘ—д»¶пјҢе°Ҳй–Җз”Ёж–јжҸҗеҚҮ .NET й–Ӣзҷјдәәе“Ўзҡ„з”ҹз”ўеҠӣгҖӮе®ғжҸҗдҫӣдәҶеј·еӨ§зҡ„зЁӢејҸзўјеҲҶжһҗгҖҒйҮҚж§Ӣе·Ҙе…·е’Ңжҷәж…§жҸҗзӨәеҠҹиғҪгҖӮ

дё»иҰҒеҠҹиғҪд»Ӣзҙ№

1. еҚіжҷӮзЁӢејҸзўјеҲҶжһҗ

  • иҮӘеӢ•еҒөжё¬жҪӣеңЁе•ҸйЎҢе’Ңж•ҲиғҪз“¶й ё
  • жҸҗдҫӣжңҖдҪіеҜҰиёҗе»әиӯ°
  • зӘҒйЎҜжңӘдҪҝз”Ёзҡ„зЁӢејҸзўје’ҢеӨҡйӨҳзҡ„еҸғз…§
  • еҚіжҷӮжӘўжҹҘе‘ҪеҗҚиҰҸеүҮз¬ҰеҗҲжҖ§

2. еј·еӨ§зҡ„йҮҚж§Ӣе·Ҙе…·

  • и®Ҡж•ёйҮҚе‘ҪеҗҚпјҲж”ҜжҸҙи·ЁжӘ”жЎҲпјү
  • ж–№жі•жҸҗеҸ–иҲҮе…§еөҢ
  • д»ӢйқўжҸҗеҸ–
  • зЁӢејҸзўјз§»еӢ•иҲҮйҮҚзө„
  • иҮӘеӢ•з”ўз”ҹе»әж§ӢеҮҪејҸгҖҒеұ¬жҖ§зӯүжЁЈжқҝзЁӢејҸзўј

3. е°ҺиҰҪеўһеј·

  • еҝ«йҖҹжҗңе°Ӣд»»дҪ•е…§е®№пјҲAlt+`)
  • и·іиҮіе®Је‘Ҡ/еҜҰдҪң
  • жүҫеҮәжүҖжңүеҸғз…§
  • жӘ”жЎҲзөҗж§ӢиҰ–ең–
  • з№јжүҝйҡҺеұӨжӘўиҰ–

4. зЁӢејҸзўјз”ўз”ҹиҲҮжЁЈжқҝ

  • иҮӘеӢ•е®ҢжҲҗеёёз”ЁзЁӢејҸзўјзүҮж®ө
  • е®ўиЈҪеҢ–зЁӢејҸзўјжЁЈжқҝ
  • е–®е…ғжё¬и©Ұз”ўз”ҹеҷЁ
  • е»әж§ӢеҮҪејҸеҸғж•ёеҢ…иЈқеҷЁ

жҺҲж¬ҠйҒёж“Ү

ReSharper жҸҗдҫӣеӨҡзЁ®жҺҲж¬Ҡж–№жЎҲпјҡ

  1. еҖӢдәәжҺҲж¬Ҡ

    • йҒ©еҗҲеҖӢдәәй–ӢзҷјиҖ…
    • еҸҜеңЁеӨҡеҸ°йӣ»и…ҰдёҠдҪҝз”Ё
    • жҸҗдҫӣжңҲд»ҳжҲ–е№ҙд»ҳйҒёй …

  2. е•ҶжҘӯжҺҲж¬Ҡ

    • йҒ©еҗҲе…¬еҸёеңҳйҡҠдҪҝз”Ё
    • жҸҗдҫӣеңҳйҡҠзүҲжң¬жҺ§з®Ў
    • еҢ…еҗ«жҠҖиЎ“ж”ҜжҸҙжңҚеӢҷ

  3. и©Ұз”ЁзүҲ

    • жҸҗдҫӣ 30 еӨ©е…ҚиІ»и©Ұз”Ё
    • еҠҹиғҪе®Ңе…Ёй–Ӣж”ҫ
    • з„Ўй ҲдҝЎз”ЁеҚЎиіҮиЁҠ

ж•ҲиғҪиҖғйҮҸ

йӣ–з„¶ ReSharper еҠҹиғҪеј·еӨ§пјҢдҪҶд№ҹйңҖиҰҒжіЁж„ҸдёҖдәӣдҪҝз”ЁдёҠзҡ„иҖғйҮҸпјҡ

  1. иЁҳжҶ¶й«”дҪҝз”Ё

    • ReSharper жңғдҪ”з”ЁијғеӨҡзі»зөұиіҮжәҗ
    • е»әиӯ°йӣ»и…Ұй…ҚеӮҷиҮіе°‘ 16GB RAM
    • еӨ§еһӢе°ҲжЎҲеҸҜиғҪйңҖиҰҒжӣҙеӨҡиіҮжәҗ

  2. и§Јжұәж–№жЎҲеӨ§е°Ҹ

    • е°ҲжЎҲи¶ҠеӨ§пјҢеҲқж¬Ўијүе…ҘжҷӮй–“и¶Ҡй•·
    • е»әиӯ°дҪҝз”Ё SSD д»ҘжҸҗеҚҮж•ҲиғҪ
    • еҸҜиҖғж…®еҸӘијүе…ҘйңҖиҰҒзҡ„е°ҲжЎҲ

дҪҝз”ЁжҠҖе·§

  1. е–„з”Ё Solution-Wide Analysis

    • еҸҜеҚіжҷӮжӘўжҹҘж•ҙеҖӢж–№жЎҲзҡ„е•ҸйЎҢ
    • йҒ©еҗҲеңЁ code review еүҚеҹ·иЎҢ
    • еҸҜжүҫеҮәи·Ёе°ҲжЎҲзҡ„зӣёдҫқжҖ§е•ҸйЎҢ

  2. е®ўиЈҪеҢ–зЁӢејҸзўјжЁЈејҸ

    • иЁӯе®ҡз¬ҰеҗҲеңҳйҡҠзҡ„з·ЁзўјиҰҸзҜ„
    • зөұдёҖж јејҸеҢ–иҰҸеүҮ
    • иҮӘеӢ•еҹ·иЎҢзЁӢејҸзўјжё…зҗҶ

  3. ж•ҙеҗҲзүҲжң¬жҺ§еҲ¶

    • ж”ҜжҸҙ Git ж“ҚдҪңж•ҙеҗҲ
    • йЎҜзӨәзЁӢејҸзўји®ҠжӣҙиЁҳйҢ„
    • иЎқзӘҒи§Јжұәиј”еҠ©

зөҗи«–

ReSharper жҳҜдёҖеҖӢеҠҹиғҪйқһеёёе®Ңж•ҙзҡ„й–Ӣзҷје·Ҙе…·пјҢйӣ–з„¶йңҖиҰҒд»ҳиІ»дҪҝз”ЁпјҢдҪҶе…¶жҸҗдҫӣзҡ„з”ҹз”ўеҠӣжҸҗеҚҮзө•е°ҚеҖјеҫ—жҠ•иіҮгҖӮе°Қж–јз¶“еёёдҪҝз”Ё Visual Studio йҖІиЎҢ .NET й–Ӣзҷјзҡ„й–ӢзҷјиҖ…дҫҶиӘӘпјҢReSharper е№ҫд№ҺжҳҜеҝ…еӮҷзҡ„е·Ҙе…·д№ӢдёҖгҖӮ
