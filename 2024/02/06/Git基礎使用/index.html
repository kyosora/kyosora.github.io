Git еҹәзӨҺж•ҷеӯё

жӯЎиҝҺдҫҶеҲ° Git зҡ„дё–з•ҢпјҒйҖҷзҜҮж–Үз« е°Үжңғеё¶й ҳдҪ еҝ«йҖҹдәҶи§Ј Git зҡ„еҹәзӨҺдҪҝз”Ёж–№жі•пјҢеҢ…жӢ¬зүҲжң¬жҺ§еҲ¶зҡ„жҰӮеҝөгҖҒеҰӮдҪ•иЁӯзҪ® GitгҖҒеҹәжң¬жҢҮд»ӨзӯүгҖӮи®“жҲ‘еҖ‘й–Ӣе§Ӣеҗ§пјҒ

Git жҳҜд»Җйәјпјҹ

Git жҳҜдёҖеҖӢй–Ӣжәҗзҡ„еҲҶж•ЈејҸзүҲжң¬жҺ§еҲ¶зі»зөұпјҢеҸҜд»Ҙжңүж•ҲгҖҒй«ҳж•Ҳең°иҷ•зҗҶеҫһе°ҸеҲ°еӨ§зҡ„й …зӣ®гҖӮGit жҳҜзӮәдәҶе„ӘеҢ–жҸҗй«ҳж•ҲзҺҮзҡ„зүҲжң¬жҺ§еҲ¶иҖҢиЁӯиЁҲзҡ„пјҢе®ғзҡ„иЁұеӨҡжҰӮеҝөеңЁе…¶д»–зүҲжң¬жҺ§еҲ¶зі»зөұдёӯйғҪжңүжүҖдёҚеҗҢгҖӮ

е®үиЈқ Git

йҰ–е…ҲпјҢдҪ йңҖиҰҒеңЁдҪ зҡ„ж©ҹеҷЁдёҠе®үиЈқ GitгҖӮGit еҸҜд»ҘеңЁеҗ„зЁ®ж“ҚдҪңзі»зөұдёҠйҒӢиЎҢпјҡ

  • е°Қж–ј WindowsпјҢдҪ еҸҜд»Ҙеҫһ Git е®ҳж–№з¶Із«ҷ дёӢијүе®үиЈқеҢ…гҖӮ
  • еңЁ macOS дёҠпјҢеҸҜд»ҘйҖҡйҒҺ Homebrew е‘Ҫд»Ө brew install git е®үиЈқгҖӮ
  • е°Қж–ј Linux з”ЁжҲ¶пјҢеҸҜд»ҘйҖҡйҒҺдҪ зҡ„еҢ…з®ЎзҗҶеҷЁе®үиЈқпјҢеҰӮ Ubuntu зҡ„ sudo apt-get install git е‘Ҫд»ӨгҖӮ

е®үиЈқйҒҺзЁӢдёӯпјҢеҹәжң¬дёҠй»ҳиӘҚйҒёй …е°ұеҫҲеӨ з”ЁдәҶпјҢдёҖи·ҜNextеҲ°еә•е°ұе°ҚдәҶгҖӮ

иЁӯзҪ® Git

е®үиЈқе®Ң Git еҫҢпјҢжү“й–ӢдҪ зҡ„зөӮз«Ҝж©ҹпјҲTerminalжҲ–Command PromptпјүпјҢз¬¬дёҖд»¶дәӢе°ұжҳҜиЁӯзҪ®дҪ зҡ„з”ЁжҲ¶еҗҚе’Ңйӣ»еӯҗйғөд»¶ең°еқҖгҖӮйҖҷеҫҲйҮҚиҰҒпјҢеӣ зӮәжҜҸж¬Ў Git жҸҗдәӨжҷӮйғҪжңғдҪҝз”ЁйҖҷдәӣдҝЎжҒҜпјҡ

жҒӯе–ңдҪ пјҒзҸҫеңЁе·Із¶“жҺҢжҸЎдәҶ Git зҡ„еҹәзӨҺжҢҮд»ӨгҖӮйҖҷдәӣжҳҜжҜҸдҪҚй–ӢзҷјиҖ…йғҪжҮүи©ІдәҶи§Јзҡ„еҹәжң¬ж“ҚдҪңпјҢжҺҘдёӢдҫҶпјҢдҫҶи«Үи«ҮpushеҸҜиғҪжңғйҒҮеҲ°зҡ„е•ҸйЎҢгҖӮ

и§ЈжұәGit PushеӨұж•—зҡ„ж–№жЎҲ

з•¶дҪ еҹ·иЎҢgit pushжҷӮйҒӯйҒҮеӨұж•—пјҢеҸҜиғҪжңғзңӢеҲ°еҗ„зЁ®йҢҜиӘӨиЁҠжҒҜгҖӮйҖҷиЈЎеҲ—иҲүе№ҫзЁ®еёёиҰӢеҺҹеӣ е’Ңе»әиӯ°зҡ„и§ЈжұәиҫҰжі•пјҡ

еҺҹеӣ дёҖпјҡйҒ зЁӢеҖүеә«жңүж–°зҡ„жӣҙж”№

еҰӮжһңйҒ зЁӢеҖүеә«пјҲremote repositoryпјүжңүж–°зҡ„жҸҗдәӨпјҢиҖҢдҪ зҡ„жң¬ең°еҖүеә«жІ’жңүйҖҷдәӣжӣҙж–°пјҢpushеҸҜиғҪжңғеӨұж•—гҖӮ

и§Јжұәж–№жЎҲпјҡ

е…ҲжӢүеҸ–пјҲPullпјүжңҖж–°зҡ„жӣҙж”№пјҡ

дҪҝз”Ёgit pullе‘Ҫд»Өе…Ҳе°ҮйҒ зЁӢеҖүеә«зҡ„жӣҙж”№жӢүеҸ–дёӢдҫҶдёҰеҗҲдҪөеҲ°дҪ зҡ„жң¬ең°еҲҶж”ҜгҖӮ

и§ЈжұәиЎқзӘҒпјҲеҰӮжһңжңүзҡ„и©ұпјүпјҡ

еҰӮжһңеңЁеҗҲдҪөпјҲmergeпјүйҒҺзЁӢдёӯеҮәзҸҫиЎқзӘҒпјҢдҪ йңҖиҰҒжүӢеӢ•и§ЈжұәйҖҷдәӣиЎқзӘҒгҖӮи§ЈжұәеҫҢпјҢеҶҚж¬ЎжҸҗдәӨжӣҙж”№гҖӮ

еҶҚж¬ЎжҺЁйҖҒпјҡ

и§ЈжұәиЎқзӘҒдёҰжҸҗдәӨеҫҢпјҢдҪ еҸҜд»ҘеҶҚж¬Ўеҳ—и©ҰжҺЁйҖҒеҲ°йҒ зЁӢеҖүеә«гҖӮ

еҺҹеӣ дәҢпјҡжІ’жңүжҺЁйҖҒж¬Ҡйҷҗ

еҰӮжһңдҪ е°ҚйҒ зЁӢеҖүеә«жІ’жңүеҜ«е…Ҙж¬ҠйҷҗпјҢдҪ зҡ„pushж“ҚдҪңжңғиў«жӢ’зө•гҖӮ

и§Јжұәж–№жЎҲпјҡ

и«ӢжұӮеӯҳеҸ–ж¬ҠйҷҗпјҡиҒҜз№«еҖүеә«зҡ„ж“ҒжңүиҖ…жҲ–з®ЎзҗҶе“ЎпјҢи«ӢжұӮзҚІеҫ—жҺЁйҖҒзҡ„ж¬ҠйҷҗгҖӮ

еҺҹеӣ дёүпјҡеҲҶж”Ҝиў«йҺ–е®ҡ

еңЁдёҖдәӣе°ҲжЎҲдёӯпјҢзү№е®ҡзҡ„еҲҶж”ҜеҸҜиғҪжңғиў«йҺ–е®ҡпјҢйҳІжӯўзӣҙжҺҘжҺЁйҖҒгҖӮ

и§Јжұәж–№жЎҲпјҡ

дҪҝз”ЁжӢүеҸ–и«ӢжұӮпјҲPull Requestпјүпјҡеүөе»әдёҖеҖӢжӢүеҸ–и«ӢжұӮпјҲPull RequestпјүпјҢйҖҡйҒҺд»ЈзўјеҜ©жҹҘеҫҢпјҢз”ұжңүж¬Ҡйҷҗзҡ„дәәеҗҲдҪөеҲ°зӣ®жЁҷеҲҶж”ҜгҖӮ

еҺҹеӣ еӣӣпјҡжҺЁйҖҒеҲ°йҢҜиӘӨзҡ„еҲҶж”Ҝ

жңүжҷӮеҖҷпјҢдҪ еҸҜиғҪдёҚе°Ҹеҝғеҳ—и©ҰжҺЁйҖҒеҲ°дёҖеҖӢдёҚеӯҳеңЁжҲ–жҳҜдҪ жІ’жңүж¬Ҡйҷҗзҡ„еҲҶж”ҜгҖӮ

и§Јжұәж–№жЎҲпјҡ

жӘўжҹҘеҲҶж”ҜеҗҚзЁұпјҡзўәдҝқдҪ жҺЁйҖҒеҲ°зҡ„еҲҶж”ҜеҗҚзЁұжҳҜжӯЈзўәзҡ„гҖӮдҪ еҸҜд»ҘдҪҝз”Ёgit branch -aжҹҘзңӢжүҖжңүжң¬ең°е’ҢйҒ зЁӢеҲҶж”ҜгҖӮ
