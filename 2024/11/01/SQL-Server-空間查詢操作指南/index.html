六、效能最佳化建議

  1. 建立適當的空間索引
  2. 優先過濾非空間條件
  3. 避免在大量資料上進行複雜的空間運算
  4. 適當使用 CTE 或子查詢來優化查詢結構
  5. 確保所有幾何物件使用相同的空間參考系統(SRID)

結語
