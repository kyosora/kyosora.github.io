解決 C 槽爆炸危機！把 Ollama 模型檔案搬到 D 槽

發表於2024-12-11|AI工具
|總字數:476|閱讀時間:1分鐘

為什麼要移動 Ollama 模型？

Ollama 是一個強大的本地端 AI 模型執行工具，但預設將所有模型檔案都存放在 C 槽，隨著下載的模型越來越多，很容易造成系統碟空間不足。本教學將帶你一步步將模型檔案移動到 D 槽，讓你的 C 槽重獲新生！

移動前的準備工作

  1. 確認 D 槽有足夠空間
  2. 先關閉所有正在運行的 Ollama 程序
  3. 備份重要的模型設定（建議）

步驟二：移動現有模型

  1. 打開檔案總管，預設模型位置在：
  1. 將 .ollama 資料夾內的所有檔案複製到 D:\Ollama

步驟三：設定環境變數

  1. 按下 Windows + R
  2. 輸入 sysdm.cpl 開啟系統內容
  3. 切換到「進階」頁籤
  4. 點選「環境變數」
  5. 在「系統變數」區域中新增：
    • 變數名稱：OLLAMA_MODELS
    • 變數值：D:\Ollama

常見問題解答

Q1：移動後模型無法使用怎麼辦？

確認環境變數是否正確設定，並重新啟動電腦試試看。

Q2：要如何確認移動成功？

檢查 D:\Ollama 資料夾的大小，應該要和原本 C 槽的 .ollama 資料夾大小相近。

Q3：移動過程中可以取消嗎？

建議一次完成所有步驟，中途取消可能會造成模型檔案不完整。

注意事項

  • 移動過程中請勿關機或重新啟動電腦
  • 確保有足夠的磁碟空間
  • 建議在移動前先備份重要的模型檔案
