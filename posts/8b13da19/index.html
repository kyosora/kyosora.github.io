自動化部署概述

完整的自動化部署流程包含以下步驟：

  1. 開發者推送程式碼到 GitHub
  2. GitHub Actions 自動執行測試和建置
  3. 建立新的 Docker 映像檔並推送至 Docker Hub
  4. 生產環境自動偵測並更新容器
  5. 健康檢查確認部署狀態

    D --> E[推送至 Docker Hub]

    F --> G[健康檢查]
    G -->|成功| H[部署完成]
    G -->|失敗| I[自動回滾]
