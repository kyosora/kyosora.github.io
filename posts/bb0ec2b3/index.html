еҫһ .NET Framework еҲ° .NET CoreпјҡдёҖеҖӢеҝ…иҰҒзҡ„иҪүи®Ҡ

.NET Framework е’Ң .NET CoreпјҲзҸҫзЁұ .NETпјүдёҚжҳҜз«¶зҲӯзҡ„жӣҝд»Је“ҒвҖ”вҖ”е®ғеҖ‘д»ЈиЎЁдёҚеҗҢдё–д»Јзҡ„иЁӯиЁҲжұәзӯ–гҖӮ.NET Framework иӘ•з”ҹж–ј 2002 е№ҙпјҢз¶Ғжӯ»еңЁ Windows дёҠпјӣ.NET Core еҫһ 2016 е№ҙиө·йҮҚж–°иЁӯиЁҲпјҢи·Ёе№іеҸ°гҖҒй–ӢжәҗгҖҒжЁЎзө„еҢ–гҖӮеҫ®и»ҹе·ІжҳҺзўәе®Је‘Ҡ .NET Framework 4.8.1 жҳҜжңҖеҫҢдёҖеҖӢдё»иҰҒзүҲжң¬пјҢдёҚеҶҚеҠ е…Ҙж–°еҠҹиғҪпјҢеҸӘеҒҡе®үе…ЁжҖ§дҝ®иЈңгҖӮйҖҷеҖӢжҷӮй–“й»һе°ҚжӯЈеңЁи©•дј°жҳҜеҗҰйҒ·з§»зҡ„еңҳйҡҠдҫҶиӘӘжҳҜжңҖеҜҰйҡӣзҡ„зҗҶз”ұгҖӮ

д»ҘдёӢеҫһеӣӣеҖӢйқўеҗ‘иӘӘжҳҺе·®з•°гҖӮ

и·Ёе№іеҸ°

.NET Core еҫһиЁӯиЁҲд№ӢеҲқе°ұд»Ҙи·Ёе№іеҸ°зӮәзӣ®жЁҷпјҢеҗҢдёҖд»ҪзЁӢејҸзўјеҸҜд»ҘеңЁ WindowsгҖҒLinux е’Ң macOS дёҠе»әзҪ®иҲҮеҹ·иЎҢгҖӮйҖҷе°ҚжғіжҠҠе·ҘдҪңиІ ијүжҗ¬дёҠ Linux е®№еҷЁжҲ–йӣІз«Ҝзҡ„еңҳйҡҠеҪұйҹҝжңҖзӣҙжҺҘпјҡе®№еҷЁеҹәеә•жҳ еғҸеҸҜд»ҘйҒёжӣҙзІҫз°Ўзҡ„ Linux зүҲжң¬пјҢдёҚеҝ…её¶и‘— Windows Server жҺҲж¬ҠиІ»гҖӮ

.NET Framework жң¬иә«еҸӘж”ҜжҸҙ WindowsгҖӮMono жҳҜеҸҰдёҖеҖӢзҚЁз«Ӣзҡ„й–Ӣжәҗ .NET еҜҰдҪңпјҲжңҖеҲқз”ұ Novell/Xamarin й–ӢзҷјпјҢ2016 е№ҙ Microsoft ж”¶иіј Xamarin еҫҢзҙҚе…Ҙз®ЎзҗҶпјүпјҢеҸҜд»Ҙи®“ C# зЁӢејҸзўји·‘еңЁ Linux е’Ң macOS дёҠпјҢдҪҶе®ғе’Ң .NET Framework жҳҜе…©еҘ—еҲҶй–Ӣзҡ„ runtimeпјҢAPI зӣёе®№зЁӢеәҰдёҰйқһзҷҫеҲҶд№ӢзҷҫпјҢйҒҮеҲ° Windows-only API д»Қз„¶жңғиё©еқ‘гҖӮеҺҹз”ҹи·Ёе№іеҸ°е’ҢгҖҢйҖҸйҒҺеҸҰдёҖеҘ— runtime й–“жҺҘзӣёе®№гҖҚжҳҜе®Ңе…ЁдёҚеҗҢзҡ„дәӢгҖӮ

ж•ҲиғҪ

.NET Core жӯ·д»ЈзүҲжң¬еңЁж•ҲиғҪдёҠжңүзі»зөұжҖ§жҠ•е…ҘпјҢеҫһ JIT з·ЁиӯҜеҷЁпјҲRyuJITпјүгҖҒSpan<T> дҪҺй…ҚзҪ® APIпјҢеҲ° ASP.NET Core зҡ„ HTTP pipeline йҮҚеҜ«пјҢйғҪжңүеҸҜйҮҸжё¬зҡ„ж”№йҖІгҖӮTechEmpower Web Framework Benchmarks зҡ„ plaintext й …зӣ®дёӯпјҢASP.NET Core й•·жңҹжҺ’еңЁеҗ„дё»жөҒжЎҶжһ¶зҡ„еүҚж®өпјҢдёҰдёҚжҳҜиЎҢйҠ·иӘһиЁҖгҖӮ

.NET Framework зҡ„ж•ҲиғҪеҸ—йҷҗж–јз•¶е№ҙзҡ„иЁӯиЁҲйҒёж“ҮпјҢеҫҲеӨҡдҪҺеұӨеҜҰдҪңйӣЈд»ҘеңЁдёҚз ҙеЈһеҗ‘дёӢзӣёе®№зҡ„еүҚжҸҗдёӢжӣҝжҸӣпјҢйҖҷд№ҹжҳҜеҫ®и»ҹжұәе®ҡеҫһй ӯйҮҚеҜ«зҡ„ж №жң¬еҺҹеӣ д№ӢдёҖгҖӮ

йғЁзҪІ

.NET Core жҸҗдҫӣе…©зЁ®дё»иҰҒйғЁзҪІжЁЎејҸпјҡ

  • SCDпјҲиҮӘеҢ…еҗ«йғЁзҪІпјүпјҡжҠҠ runtime е’ҢжҮүз”ЁзЁӢејҸдёҖиө·жү“еҢ…гҖӮзӣ®жЁҷж©ҹеҷЁдёҚйңҖиҰҒй җиЈқ .NETпјҢеҸҜд»Ҙе®Ңе…ЁжҺ§еҲ¶жүҖз”Ёзҡ„ runtime зүҲжң¬пјҢд»Јеғ№жҳҜз”ўеҮәзү©ијғеӨ§гҖӮ
  • FDDпјҲжЎҶжһ¶дҫқиіҙйғЁзҪІпјүпјҡеҸӘжү“еҢ…жҮүз”ЁзЁӢејҸжң¬иә«пјҢзӣ®жЁҷж©ҹеҷЁеҝ…й Ҳй җиЈқе°ҚжҮүзүҲжң¬зҡ„ .NET runtimeгҖӮз”ўеҮәзү©е°ҸпјҢдҪҶз’°еўғеҝ…й Ҳе…ҲеӮҷеҰҘгҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҡFDD дёҚжҳҜгҖҢдёҚеҝ…ж“”еҝғзӣ®жЁҷз’°еўғзҡ„ .NET зүҲжң¬гҖҚвҖ”вҖ”жҒ°жҒ°зӣёеҸҚпјҢFDD й«ҳеәҰдҫқиіҙзӣ®жЁҷж©ҹеҷЁзҡ„ runtime зүҲжң¬гҖӮеҸӘжңү SCD жүҚиғҪе®Ңе…Ёйҡ”йӣўз’°еўғе·®з•°гҖӮ.NET 5 д№ӢеҫҢйӮ„еҠ е…ҘдәҶ Single-file publishпјҲж•ҙеҖӢжҮүз”Ёжү“жҲҗе–®дёҖеҹ·иЎҢжӘ”пјүе’Ң ReadyToRunпјҲй җз·ЁиӯҜйҷҚдҪҺеҶ·е•ҹеӢ•жҷӮй–“пјүзӯүйҒёй …пјҢеҸҜд»ҘиҰ–йңҖжұӮз–ҠеҠ дҪҝз”ЁгҖӮ

.NET Framework зҡ„йғЁзҪІз¶ҒеңЁ Windows Update е’Ңзі»зөұй җиЈқзҡ„ .NET Framework зүҲжң¬дёҠпјҢеҚҮзүҲжң¬жҷӮеҫҖеҫҖйңҖиҰҒдҫқиіҙдҪңжҘӯзі»зөұзҡ„жӣҙж–°йҖұжңҹпјҢеҪҲжҖ§ијғдҪҺгҖӮ

й–Ӣзҷјз”ҹж…ӢиҲҮеҫҢзәҢз¶ӯиӯ·

.NET Core жҳҜ Microsoft зӣ®еүҚзҡ„дё»з·ҡй–Ӣзҷје№іеҸ°пјҢж–°зүҲ C# иӘһиЁҖзү№жҖ§гҖҒж–°зҡ„ BCL APIгҖҒж•ҲиғҪж”№йҖІйғҪеҸӘеңЁйҖҷжўқз·ҡдёҠеҮәзҸҫгҖӮ.NET Framework з¶ӯжҢҒеңЁ 4.8.1пјҢеҫ®и»ҹзҡ„е®ҳж–№з«Ӣе ҙжҳҜпјҡеҸӘеҒҡе®үе…ЁжҖ§дҝ®иЈңпјҢдёҚеҠ еҠҹиғҪгҖӮ

й•·жңҹдҫҶзңӢпјҢз•ҷеңЁ .NET Framework зӯүж–јжҺҘеҸ—жҠҖиЎ“еҮҚзөҗгҖӮеҘ—д»¶з”ҹж…Ӣд№ҹеңЁеҫҖ .NET 5+ еӮҫж–ңпјҢйғЁеҲҶ NuGet еҘ—д»¶е·ІеҒңжӯўзҷјеёғж”ҜжҸҙ .NET Framework зҡ„зүҲжң¬гҖӮ

е“ӘдәӣжҮүз”ЁйҒ·з§»жҜ”ијғйә»з…©

дёҰйқһжүҖжңү .NET Framework жҮүз”ЁйғҪиғҪй ҶеҲ©жҗ¬йҒҺеҺ»гҖӮе№ҫйЎһе…ёеһӢзҡ„й«ҳйӣЈеәҰжғ…жіҒпјҡ

  • WCF жңҚеӢҷз«Ҝпјҡ.NET дёҠжІ’жңүе…§е»әзҡ„ WCF жңҚеӢҷз«ҜеҜҰдҪңгҖӮCoreWCF жҳҜзӨҫзҫӨз¶ӯиӯ·зҡ„жӣҝд»Јж–№жЎҲпјҢиҰҶи“ӢзҺҮдёҚе®Ңж•ҙгҖӮ
  • WebFormsпјҡжһ¶ж§Ӣе·®з•°еӨӘеӨ§пјҢжІ’жңүе°ҚжҮүзҡ„ .NET Core зүҲжң¬пјҢеҸӘиғҪж”№еҜ«гҖӮ
  • WinForms / WPFпјҡ.NET 5+ иө·жӯЈејҸж”ҜжҸҙпјҢдҪҶеҝ…й Ҳи·‘еңЁ Windows дёҠпјҢи·Ёе№іеҸ°е„ӘеӢўеңЁжӯӨжғ…еўғдёҚйҒ©з”ЁгҖӮ
  • COM Interop / Windows зү№е®ҡ APIпјҡд»Қз„¶еҸҜд»Ҙи·‘пјҢдҪҶжңүдәӣиЎҢзӮәе’Ң .NET Framework дёҚе®Ңе…ЁдёҖиҮҙгҖӮ

и©•дј°йҒ·з§»жҷӮпјҢеҫ®и»ҹзҡ„ .NET Upgrade Assistant еҸҜд»ҘжҺғжҸҸзҸҫжңүе°ҲжЎҲгҖҒжЁҷеҮәзӣёе®№жҖ§е•ҸйЎҢпјҢзөҰеҮәйҒ·з§»йӣЈеәҰзҡ„еҲқжӯҘдј°иЁҲгҖӮе·Ҙе…·жң¬иә«дёҚиғҪе…ЁиҮӘеӢ•е®ҢжҲҗйҒ·з§»пјҢдҪҶеҸҜд»ҘзңҒжҺүеҫҲеӨҡжүӢеӢ•жҺ’жҹҘзҡ„жҷӮй–“гҖӮ
