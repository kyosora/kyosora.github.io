NSSMпјҲNon-Sucking Service ManagerпјүжҳҜдёҖеҖӢеј·еӨ§зҡ„е·Ҙе…·пјҢеҸҜд»Ҙиј•й¬Ҷең°е°Үд»»дҪ•еҸҜеҹ·иЎҢж–Үд»¶пјҲEXEпјүиҪүжҸӣзӮә Windows жңҚеӢҷгҖӮжң¬ж•ҷеӯёе°ҮжҢҮе°ҺжӮЁе®ҢжҲҗж•ҙеҖӢйҒҺзЁӢпјҢеҫһе®үиЈқ NSSM еҲ°й…ҚзҪ®е’Ңз®ЎзҗҶжӮЁзҡ„жңҚеӢҷгҖӮ

зӣ®йҢ„

  1. NSSM з°Ўд»Ӣ
  2. дёӢијүе’Ңе®үиЈқ NSSM
  3. еҹәжң¬дҪҝз”Ёпјҡеүөе»әжңҚеӢҷ
  4. й«ҳзҙҡй…ҚзҪ®йҒёй …
  5. з®ЎзҗҶжңҚеӢҷ
  6. еёёиҰӢе•ҸйЎҢи§Јжұә
  7. жңҖдҪіеҜҰиёҗе’ҢжіЁж„ҸдәӢй …

1. NSSM з°Ўд»ӢиҲҮSCе‘Ҫд»Өзҡ„жҜ”ијғ

NSSM жҳҜдёҖеҖӢе…ҚиІ»зҡ„й–Ӣжәҗе·Ҙе…·пјҢе®ғе…ҒиЁұжӮЁе°Үе№ҫд№Һд»»дҪ•жҮүз”ЁзЁӢеәҸдҪңзӮә Windows жңҚеӢҷйҒӢиЎҢгҖӮиҲҮSCе‘Ҫд»ӨжҜ”ијғ

SCе‘Ҫд»Өзҡ„е„Әй»һпјҡ

1. Windows е…§е»әпјҢз„ЎйңҖйЎҚеӨ–е®үиЈқ

2. е‘Ҫд»Өз°Ўе–®пјҢйҒ©еҗҲеҹәжң¬йңҖжұӮ

SCе‘Ҫд»Өзҡ„йҷҗеҲ¶пјҡ

1. еҠҹиғҪзӣёе°Қз°Ўе–®пјҢзјәд№Ҹй«ҳзҙҡйҒёй …

2. е°Қж–јиӨҮйӣңзҡ„е•ҹеӢ•еҸғж•ёжҲ–з’°еўғиЁӯзҪ®пјҢеҸҜиғҪйңҖиҰҒйЎҚеӨ–зҡ„и…іжң¬

3. йҢҜиӘӨиҷ•зҗҶе’Ңж—ҘиӘҢиЁҳйҢ„еҠҹиғҪжңүйҷҗ

NSSMзҡ„е„Әй»һпјҡ

1. ең–еҪўд»Ӣйқўпјҡжҳ“ж–јдҪҝз”ЁпјҢдёҚйңҖиҰҒиЁҳжҶ¶иӨҮйӣңзҡ„е‘Ҫд»ӨеҲ—иӘһжі•

2. иұҗеҜҢзҡ„зө„ж…ӢйҒёй …пјҡ

- еҸҜд»ҘиЁӯзҪ®е•ҹеӢ•зӣ®йҢ„гҖҒз’°еўғи®ҠйҮҸ

- ж”ҜжҸҙиӨҮйӣңзҡ„е•ҹеӢ•еҸғж•ё

- еҸҜд»Ҙзө„ж…ӢжңҚеӢҷдҫқиіҙй—ңдҝӮ

3. йҢҜиӘӨиҷ•зҗҶпјҡеҸҜд»ҘиЁӯзҪ®еңЁзЁӢеәҸеҙ©жҪ°жҷӮиҮӘеӢ•йҮҚе•ҹ

4. ж—ҘиӘҢеҠҹиғҪпјҡеҸҜд»ҘжҚ•зҚІдёҰиЁҳйҢ„жЁҷжә–ијёеҮәе’ҢйҢҜиӘӨијёеҮә

5. й—ңж©ҹиЎҢзӮәжҺ§еҲ¶пјҡеҸҜд»ҘиЁӯзҪ®еҰӮдҪ•е„Әйӣ…ең°й—ңй–үжңҚеӢҷ

6. I/O йҮҚж–°е°Һеҗ‘пјҡеҸҜд»Ҙе°Үијёе…ҘијёеҮәйҮҚж–°е°Һеҗ‘еҲ°жӘ”жЎҲ

7. ж”ҜжҸҙдә’еӢ•ејҸзЁӢеәҸпјҡжҹҗдәӣйңҖиҰҒз”ЁжҲ¶ијёе…Ҙзҡ„зЁӢеәҸд№ҹеҸҜд»ҘдҪңзӮәжңҚеӢҷйҒӢиЎҢ

8. жңҚеӢҷзӣЈжҺ§пјҡжҸҗдҫӣдәҶжӣҙеҘҪзҡ„ж–№ејҸдҫҶзӣЈжҺ§жңҚеӢҷзӢҖж…Ӣ

зёҪзҡ„дҫҶиӘӘпјҢйӣ–з„¶ SC е‘Ҫд»Өе°Қж–јз°Ўе–®зҡ„йңҖжұӮдҫҶиӘӘе·Із¶“и¶іеӨ пјҢдҪҶ NSSM жҸҗдҫӣдәҶжӣҙеӨҡзҡ„йқҲжҙ»жҖ§е’ҢеҠҹиғҪпјҢзү№еҲҘйҒ©еҗҲйӮЈдәӣйңҖиҰҒжӣҙзІҫзҙ°жҺ§еҲ¶зҡ„иӨҮйӣңе ҙжҷҜгҖӮеҰӮжһңжӮЁзҡ„зЁӢеәҸйңҖиҰҒзү№ж®Ҡзҡ„е•ҹеӢ•жўқд»¶гҖҒйҢҜиӘӨиҷ•зҗҶжҲ–ж—ҘиӘҢиЁҳйҢ„пјҢNSSM еҸҜиғҪжҳҜжӣҙеҘҪзҡ„йҒёж“ҮгҖӮ

2. дёӢијүе’Ңе®үиЈқ NSSM

  1. иЁӘе•Ҹ NSSM е®ҳж–№з¶Із«ҷпјҡhttps://nssm.cc/download
  2. дёӢијүжңҖж–°зүҲжң¬зҡ„ NSSMпјҲйҒёж“ҮйҒ©еҗҲжӮЁзі»зөұзҡ„зүҲжң¬пјҢ32 дҪҚжҲ– 64 дҪҚпјү
  3. и§ЈеЈ“дёӢијүзҡ„еЈ“зё®ж–Үд»¶
  4. е°Ү nssm.exe ж”ҫзҪ®еңЁдёҖеҖӢдҫҝж–јиЁӘе•Ҹзҡ„дҪҚзҪ®пјҢеҰӮ C:\Program Files\NSSM\
  5. еҸҜйҒёпјҡе°Ү NSSM зҡ„и·Ҝеҫ‘ж·»еҠ еҲ°зі»зөұзҡ„ PATH з’°еўғи®ҠйҮҸдёӯ

3. еҹәжң¬дҪҝз”Ёпјҡеүөе»әжңҚеӢҷ

д»Ҙз®ЎзҗҶе“Ўиә«д»Ҫжү“й–Ӣе‘Ҫд»ӨжҸҗзӨәз¬ҰпјҢз„¶еҫҢжҢүз…§д»ҘдёӢжӯҘй©ҹж“ҚдҪңпјҡ

  1. еүөе»әж–°жңҚеӢҷпјҡ

    йҖҷе°Үжү“й–Ӣ NSSM зҡ„ең–еҪўз•ҢйқўгҖӮ

  2. еңЁеҪҲеҮәзҡ„зӘ—еҸЈдёӯпјҢиЁӯзҪ®д»ҘдёӢе…§е®№пјҡ

    • Path: жӮЁзҡ„ EXE ж–Үд»¶зҡ„е®Ңж•ҙи·Ҝеҫ‘
    • Startup directory: EXE ж–Үд»¶жүҖеңЁзҡ„зӣ®йҢ„
    • Arguments: еҰӮжһңйңҖиҰҒпјҢж·»еҠ е‘Ҫд»ӨиЎҢеҸғж•ё

  3. й»һж“Ҡ "Install service" жҢүйҲ•

жӮЁд№ҹеҸҜд»ҘдҪҝз”Ёе‘Ҫд»ӨиЎҢзӣҙжҺҘеүөе»әжңҚеӢҷпјҡ

4. й«ҳзҙҡй…ҚзҪ®йҒёй …

NSSM жҸҗдҫӣдәҶиЁұеӨҡй«ҳзҙҡй…ҚзҪ®йҒёй …пјҡ

  1. иЁӯзҪ®жңҚеӢҷйЎҜзӨәеҗҚзЁұе’ҢжҸҸиҝ°пјҡ

  2. й…ҚзҪ®е•ҹеӢ•йЎһеһӢпјҡ

  3. иЁӯзҪ®з’°еўғи®ҠйҮҸпјҡ

  4. й…ҚзҪ®йҢҜиӘӨиҷ•зҗҶпјҡ

  5. иЁӯзҪ®ж—ҘиӘҢж–Үд»¶пјҡ

  1. е•ҹеӢ•жңҚеӢҷпјҡ

  2. еҒңжӯўжңҚеӢҷпјҡ

  3. йҮҚе•ҹжңҚеӢҷпјҡ

  4. жҹҘзңӢжңҚеӢҷзӢҖж…Ӣпјҡ

  5. з·ЁијҜзҸҫжңүжңҚеӢҷпјҡ

  6. еҲӘйҷӨжңҚеӢҷпјҡ

6. еёёиҰӢе•ҸйЎҢи§Јжұә

  1. жңҚеӢҷз„Ўжі•е•ҹеӢ•

    • жӘўжҹҘ EXE и·Ҝеҫ‘жҳҜеҗҰжӯЈзўә
    • зўәдҝқ NSSM е’ҢжӮЁзҡ„жҮүз”ЁзЁӢеәҸжңүи¶іеӨ зҡ„ж¬Ҡйҷҗ
    • жҹҘзңӢ Windows дәӢд»¶ж—ҘиӘҢе’Ң NSSM ж—ҘиӘҢж–Үд»¶

  2. жңҚеӢҷе•ҹеӢ•еҫҢз«ӢеҚіеҒңжӯў

    • жӘўжҹҘжӮЁзҡ„жҮүз”ЁзЁӢеәҸжҳҜеҗҰиЁӯиЁҲзӮәй•·жҷӮй–“йҒӢиЎҢ
    • зўәдҝқжүҖжңүеҝ…иҰҒзҡ„дҫқиіҙй …йғҪеҸҜз”Ё

  3. з„Ўжі•иҲҮжңҚеӢҷйҖҡдҝЎ

    • жӘўжҹҘйҳІзҒ«зүҶиЁӯзҪ®
    • зўәдҝқжңҚеӢҷеёіжҲ¶жңүйҒ©з•¶зҡ„з¶ІзөЎж¬Ҡйҷҗ

7. жңҖдҪіеҜҰиёҗе’ҢжіЁж„ҸдәӢй …

  1. е§ӢзөӮдҪҝз”Ёз®ЎзҗҶе“Ўж¬ҠйҷҗйҒӢиЎҢ NSSM е‘Ҫд»Ө
  2. зӮәжӮЁзҡ„жңҚеӢҷдҪҝз”ЁжҸҸиҝ°жҖ§зҡ„еҗҚзЁұ
  3. е®ҡжңҹжӘўжҹҘжңҚеӢҷж—ҘиӘҢд»ҘзўәдҝқжӯЈеёёйҒӢиЎҢ
  4. иҖғж…®дҪҝз”ЁжңҚеӢҷеёіжҲ¶иҖҢдёҚжҳҜзі»зөұеёіжҲ¶дҫҶйҒӢиЎҢжңҚеӢҷ
  5. е®ҡжңҹжӣҙж–° NSSM еҲ°жңҖж–°зүҲжң¬
  6. еңЁз”ҹз”ўз’°еўғдёӯйғЁзҪІд№ӢеүҚпјҢеңЁжё¬и©Ұз’°еўғдёӯеҫ№еә•жё¬и©ҰжӮЁзҡ„жңҚеӢҷй…ҚзҪ®
